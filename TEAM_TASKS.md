# Encrypted Diary App - Team Task Distribution

## Project Overview
This is a secure, client-side encrypted diary application built with Next.js, featuring AES-GCM encryption, NextAuth authentication, and a cyberpunk UI theme. The core functionality has been implemented by **Sudais Khan** (Lead Developer), with remaining tasks focused on enhancements, testing, and deployment.
